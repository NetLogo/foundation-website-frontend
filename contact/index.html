Getting help

See our Resources page for guidance on where to seek help using NetLogo.

Contacting Us

Feedback from users has been very valuable to us in designing and improving NetLogo. We'd like to hear from you.

Feedback and suggestions

If you have general feedback or suggestions for the NetLogo team, email feedback@ccl.northwestern.edu or post them on the netlogo-users group.

Reporting a bug

Before reporting a bug, you may wish to check the NetLogo FAQ and our known issues on GitHub to see if it is a known problem.

To report a bug in NetLogo, email bugs@ccl.northwestern.edu or open an issue on GitHub.

Please include as much of the following information as you can:

  • As complete a description as possible of what the problem is and how it occurred.
  • The NetLogo model or code that you are having trouble with. Attach a complete model if possible.
  • Your system information: NetLogo version, OS version, and so on. This information is available from NetLogo's "About NetLogo" menu item.
  • The exact, full text of any error messages that appeared.

Contacting NetLogo Developers

NetLogo development is hosted on GitHub. Development discussion is on netlogo-devel.
